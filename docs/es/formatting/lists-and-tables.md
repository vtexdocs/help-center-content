Las tablas y listas presentan contenido estructurado, pero a veces no está claro cuándo usar una u otra. Consulta a continuación algunas reglas que puedes seguir al crearlas:

<b>Listas</b>
Usa las listas para artículos individuales o artículos con descripciones largas. Usa una lista con viñetas para elementos con un tema común que no necesitan un orden específico. Utiliza una lista numerada para pasos que deban seguirse en orden o cuando necesites priorizar elementos, como instrucciones.

<b>Tablas</b>
Usa tablas para elementos con tres o más puntos de datos relacionados. Evita usar tablas para listas simples o unidimensionales.

## Reglas generales

Cuando un elemento contiene dos puntos, la primera palabra después debe estar en minúscula.

| Correcto ✅ | Incorrecto ❌ |
| :--- | :--- |
| <ul><li><b>Reserva autorizada:</b> después de realizar el pedido.</li><li><b>Reserva confirmada:</b> después de aprobado el pago.</li></ul> | <ul><li><b>Reserva autorizada:</b> Después de realizar el pedido.</li><li><b>Reserva confirmada:</b> Después de aprobado el pago.</li></ul> |

Para garantizar que la estructura de la lista sea coherente, comienza todos los elementos con la misma forma verbal o sustantivo.

| Correcto ✅ | Incorrecto ❌ |
| :--- | :--- |
| <b>Ejemplo 1:</b><ul><li><b>Controla</b> los resultados de búsqueda utilizando condiciones específicas.</li><li><b>Visualiza</b> cómo se mostrarán los resultados de la búsqueda a los clientes en la tienda.</li><li><b>Cambia</b> el orden de los ítems listados como resultados de búsqueda.</li></ul><b>Ejemplo 2:</b><ul><li><b>Indica</b> el tipo de edición utilizado en la regla de comercialización.</li><li><b>Activa o desactiva</b> una regla de merchandising.</li></ul> O <ul><li><b>Tipo</b> edición utilizada en la regla de merchandising.</li><li><b>Opción</b> que activa o desactiva una regla de merchandising.</li></ul> | <b>Ejemplo 1:</b><ul><li><b>Controla</b> los resultados de búsqueda utilizando condiciones específicas.</li><li><b>Visualizar</b> cómo se mostrarán los resultados de la búsqueda a los clientes en la tienda.</li><li><b>Puedes cambiar</b> el orden de los ítems listados como resultados de búsqueda.</li></ul><b>Ejemplo 2:</b><ul><li><b>Indica</b> el tipo de edición utilizado en la regla de merchandising.</li><li><b>Opción</b> que activa o desactiva una regla de merchandising.</li></ul> |

Opta por infinitivo cuando los elementos de la lista comiencen con un verbo. No utilices el gerundio.

| Correcto ✅ | Incorrecto ❌ |
| :--- | :--- |
| Este artículo se divide en las siguientes secciones:<ul><li>Agregar productos</li><li>Editar productos</li></ul> | Este artículo se divide en las siguientes secciones:<ul><li>Agregando productos</li><li>Editando productos</li></ul> |

Utiliza dos puntos si la lista sigue inmediatamente a la frase introductoria.

| Correcto ✅ | Incorrecto ❌ |
| :--- | :--- |
| <ul><li>Para personalizar los botones, sigue los pasos a continuación:</li><li>Sigue los pasos a continuación para crear un archivo:</li></ul> | <ul><li>Sigue los pasos a continuación.</li><li>Crea un archivo.</li></ul> |

Utiliza un punto si hay más elementos, como un callout, entre la introducción y la lista.

| Correcto ✅ | Incorrecto ❌ |

| La competencia de promociones ocurre en las siguientes situaciones:<ul><li>Promociones con el mismo tipo de descuento.</li><li>Más de una promoción aplicada al mismo ítem.</li></ul> | La competencia de promociones ocurre en las siguientes situaciones.<ul><li>Promociones con el mismo tipo de descuento</li><li>Más de una promoción aplicada al mismo ítem</li></ul> |

## Listas

### Listas ordenadas

Opta por listas ordenadas (numeradas) cuando el orden de los elementos sea relevante, como en instrucciones paso a paso, secuencias cronológicas o procesos.

| Correcto ✅ | Incorrecto ❌ |
| :--- | :--- |
| Para crear una regla de divergencia de precios, sigue los pasos a continuación:<ol><li>En el Admin VTEX, accede a <b>Configuración de la tienda > Pedidos > Autorización de pedidos</b>, o escribe <b>Autorización de pedidos</b> en la barra de búsqueda en la parte superior de la página.</li><li>En la sección <i>Divergencia de precios</i>, haz clic en <code>CREAR REGLAS</code>.</li><li>En la nueva página que se abre, haz clic en <code>CREAR REGLAS</code>.</li></ol> | Para crear una regla de divergencia de precios, sigue los pasos a continuación:<ul><li>En el Admin VTEX, accede a <b>Configuración de la tienda > Pedidos > Autorización de pedidos</b>, o escribe <b>Autorización de pedidos</b> en la barra de búsqueda en la parte superior de la página.</li><li>En la sección <i>Divergencia de precios</i>, haz clic en <code>CREAR REGLAS</code>.</li><li>En la nueva página que se abre, haz clic en <code>CREAR REGLAS</code>.</li></ul> |

## Tablas

Usa una tabla si cada elemento tiene tres o más datos relacionados. Por ejemplo, para correlacionar nombres, tipos de datos y descripciones de elementos.

Los encabezados de columna y el contenido de las celdas deben comenzar con letra mayúscula, sin emplear mayúsculas sostenidas. Si la celda contiene una frase completa, termina con punto final.
