Ahora puedes exportar los hallazgos de [Security Monitor](https://help.vtex.com/es/tutorial/security-monitor--5LOVNLrrtmgSj99pM1NS4x) a un archivo CSV. Esta nueva funcionalidad brinda una mayor autonomía para registrar, compartir y profundizar en el análisis de los riesgos de seguridad detectados en tu cuenta.

## ¿Qué cambió?

Agregamos la opción de exportar hallazgos directamente desde el dashboard de Security Monitor. Ahora, puedes generar un archivo CSV con los hallazgos mostrados en la interfaz, facilitando el uso y análisis de esta información fuera de la plataforma VTEX. Sigue los pasos a continuación:

Accede a **Dashboards > Security Monitor > Security Monitor Dashboard** y haz clic en `Exportar` en la esquina superior derecha de la pantalla. Puedes seleccionar los hallazgos que deseas incluir en el archivo exportado: *Todos los hallazgos*, *Abiertos*, *Pospuestos* o *Finalizados*. Para más información, consulta [Security Monitor](https://help.vtex.com/es/tutorial/security-monitor--5LOVNLrrtmgSj99pM1NS4x).

![export-security-monitor-es](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/es/announcements/2025/exporta-hallazgos-de-security-monitor-para-agilizar-analisis-y-auditoria_1.gif)

## ¿Por qué realizamos este cambio?

Esta mejora facilita el monitoreo de hallazgos de seguridad y la integración con herramientas externas de monitoreo o auditoría.

## ¿Qué se necesita hacer?

La actualización ya está disponible para todas las cuentas que utilizan Security Monitor. No se requiere ninguna acción.
