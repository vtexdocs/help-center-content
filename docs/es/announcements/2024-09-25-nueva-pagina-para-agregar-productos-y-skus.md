La página de detalles de productos y SKUs ha sido rediseñada con el objetivo de optimizar tu experiencia para agregar, ver y editar productos y SKUs. La nueva página está disponible en beta.

![Nueva página para agregar productos y SKUs 1](https://images.ctfassets.net/alneenqid6w5/3jFNFHsyJmp5XnvSFAwa5t/7153f0142407f32ed55c6f8ec6426bf7/1._Overview-EN.gif)

## ¿Qué cambió?

Las nuevas páginas se cargan más rápido y ofrecen una experiencia más intuitiva, además de incluir una gestión de imágenes simplificada.

![Nueva página para agregar productos y SKUs 2](//images.ctfassets.net/alneenqid6w5/btC7OOSoj2OXGzpuXUcXG/e6a5ad3bb0f3e03dd740488121a5a15d/2._Images-EN.gif)

Ahora, todos los SKU de un mismo producto se agrupan en una sola página, lo que facilita la edición de múltiples SKU.

![Nueva página para agregar productos y SKUs 3](//images.ctfassets.net/alneenqid6w5/V3S6TCUjnz2UoDhIZyyN7/44a105b7bd159e876bedb01938d2176b/3._SKU-Single_Place.gif)

Además, es más fácil seleccionar categorías y marcas.

![Nueva página para agregar productos y SKUs 4](//images.ctfassets.net/alneenqid6w5/WJqYPXsk8WMffzPtisCp1/daa335612b708a7d9f259b0d658badf6/4._Navigation-EN.gif)

>ℹ️ Aprende más sobre las nuevas páginas de [producto](https://help.vtex.com/es/tutorial/adding-products-beta--29IkdEu6GofCFlltsZh2H8) y [SKU](https://help.vtex.com/es/tutorial/cadastro-de-skus-beta--4ryZ6J45kwn3jDiQBxGiiN), respectivamente.

## ¿Qué se necesita hacer?

No es necesaria ninguna acción. Todas las tiendas VTEX migrarán gradualmente a la nueva experiencia a lo largo de los próximos meses.

Si tu tienda aún no ha sido migrada, pero quieres unirte a la nueva experiencia, abre un ticket vía [soporte VTEX](https://help.vtex.com/es/support).
