Actualizamos el aspecto y las nomenclaturas de la plataforma Weni by VTEX, que ahora se llama **VTEX Agentic CX Platform**. Esta actualización, implementada el 26 de marzo, ofrece una experiencia más integrada y coherente con el ecosistema VTEX.

## ¿Qué cambió?

El principal cambio es el nombre de la plataforma, que ahora se llama **VTEX Agentic CX Platform**.

La estructura del menú también se ha reorganizado, y los elementos ahora están agrupados en dos secciones principales: **Agent Builder** y **Operaciones**.

Otras secciones también cambiaron de nombre y de posición en el menú. A continuación, detallamos el nombre anterior de la herramienta, el nombre nuevo y dónde puedes encontrarla:

| **Nombre antiguo**                       | **Nombre nuevo**            | **Ubicación**                    |
| ---------------------------------------- | --------------------------- | -------------------------------- |
| Insights                                 | **Analytics**               | En la página de inicio           |
| Agentes                                  | **Mis agentes**             | En la sección Agent Builder      |
| Build — contenido                        | **Base de conocimiento**    | En la sección Agent Builder      |
| Build— configuración de manager y perfil | **Editar manager**          | En la página **Mis agentes**     |
| Build — configuración de instrucciones   | **Editar instrucciones**    | En la página **Mis agentes**     |
| Flujos                                   | **Flujo de automatización** | En la sección Agent Builder      |
| Soporte humano                           | **Live Desk**               | En la sección Operaciones        |
| Conversaciones                           | **Auditoría**               | En la sección Operaciones        |
| Apps                                     | **Canales**                 | En la página **Configuraciones** |
| Estudio                                  | **Contactos**               | En la sección Operaciones        |
| Enviar plantillas                        | **Campañas**                | En la página **Contactos**       |

## ¿Por qué realizamos este cambio?

Desarrollamos esta actualización para mantener la consistencia visual con la identidad VTEX, unificando la experiencia del usuario en diferentes productos VTEX. Las principales ventajas son:

- **Consistencia visual:** interfaz alineada al estándar e identidad de VTEX, proporcionando una experiencia unificada.
- **Navegación más intuitiva:** el nuevo agrupamiento por secciones permite encontrar las herramientas adecuadas más rápido.
- **Nomenclaturas más accesibles:** los nuevos nombres de las secciones reflejan mejor la función de cada área, reduciendo la curva de aprendizaje.

## ¿Qué se necesita hacer?

No es necesaria ninguna acción. La actualización es automática y ya está disponible para todos los usuarios.

Para saber más sobre la plataforma VTEX Agentic CX Platform y sus funcionalidades, consulta nuestra [documentación](https://help.vtex.com/es/docs/tutorials/weni-by-vtex-categoria).
