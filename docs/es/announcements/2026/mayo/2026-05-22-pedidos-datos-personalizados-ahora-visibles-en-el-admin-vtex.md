Los datos personalizados ahora se muestran en los pedidos en el Admin de VTEX. Con esta actualización, puedes ver los campos personalizados de cada pedido y la información específica de las aplicaciones directamente en la página de detalles del pedido.

![Datos personalizados en el Admin VTEX](https://cdn.statically.io/gh/vtexdocs/help-center-content/refs/heads/main/docs/{{locale}}/docs/es/announcements/2026/mayo/2026-05-22-pedidos-datos-personalizados-ahora-visibles-en-el-admin-vtex.png)

## ¿Qué cambió?

Antes, los datos personalizados asociados a pedidos en el OMS, incluyendo los campos personalizados de la tienda (`customFields`) y los payloads por cada aplicación (customApps), solo podían consultarse mediante llamadas de API. Ahora, cuando un pedido contiene datos personalizados, tanto la página de detalles del pedido como la vista de impresión en el Admin de VTEX muestran esta información en menús desplegable: uno para los campos personalizados y otro para los payloads de las aplicaciones.

> ℹ️ Para más información sobre `customFields` y `customApps`, lee el artículo [VTEX Data Pipeline Beta](https://help.vtex.com/es/docs/tutorials/pedidos-data-pipeline-beta).

## ¿Por qué realizamos este cambio?

Las principales ventajas de este cambio son:

- Vista completa y rápida de datos personalizados directamente en la interfaz de Admin VTEX, sin necesidad de consultas manuales a la API.
- Mejor visibilidad de información específica en un solo lugar.
- Datos mostrados de forma organizada en grupos que se pueden contraer, facilitando la lectura y la gestión de los pedidos.

## ¿Qué se necesita hacer?

Para tener acceso a esta optimización, ponte en contacto con el [Soporte VTEX](https://supporticket.vtex.com/support) y solicita la activación de este recurso para tu cuenta.
