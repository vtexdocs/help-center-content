El módulo [Capacidad operativa](/es/docs/tutorials/capacidad-operativa) ya está disponible para todas las tiendas VTEX. Esta funcionalidad ha sido diseñada para ayudarte a gestionar la capacidad de los sellers para recibir y preparar pedidos. Con esta herramienta, puedes monitorear la distribución de pedidos entre los sellers y realizar ajustes para optimizar el flujo operativo, promoviendo una red logística más eficiente.

![operational_capacity_image_1_ES](https://cdn.statically.io/gh/vtexdocs/help-center-content/refs/heads/main/docs/es/tutorials/envío/capacidad-operativa/capacidad-operativa_1.png)

## ¿Qué cambió?

Anteriormente disponible solo en fase beta, el módulo Capacidad operativa ha sido optimizado significativamente a nivel de infraestructura. Entre las mejoras, destaca la posibilidad de segmentar la capacidad operativa por [política comercial](/es/docs/tutorials/como-funciona-una-politica-comercial), lo que permite una gestión aún más flexible y eficiente.

## ¿Qué se necesita hacer?

No es necesaria ninguna acción, ya que las tiendas VTEX de todas las regiones ya han sido migradas a la versión disponible globalmente. Para comenzar a utilizar la funcionalidad, activa el módulo en Admin VTEX en **Envío > Capacidad operativa**.

> ⚠️ El requisito para usar la funcionalidad es tener [cuentas franquicia](/es/docs/tutorials/que-es-una-cuenta-franquicia) asociadas a la [cuenta principal](/es/docs/tracks/cuentas-y-arquitectura).

Para más detalles, consulta el artículo [Capacidad operativa](/es/docs/tutorials/capacidad-operativa).
