## Definiciones

Una Colección es la agrupación de varios SKUs, que pueden tener características en común o no, para presentarlos en las páginas de productos de su tienda.

Normalmente, se puede agrupar SKUs por categoría o marca. Sin embargo, se pueden crear colecciones para asociar SKUs utilizando otros criterios de agrupación, como fechas conmemorativas, lanzamientos o productos más vendidos. 

Por ejemplo, usted puede crear una página para celebrar el Día de los enamorados y hacer que muestre una lista de regalos adecuados para esta ocasión creando una colección de "regalos".
