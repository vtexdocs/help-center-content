**VTEX Data Pipeline**, disponible exclusivamente para clientes VTEX, es una herramienta diseñada para explorar y analizar los datos de la tienda online almacenados en nuestros servidores. **Data Pipeline** es un producto add-on de VTEX que integra los datos del sitio web de tu tienda online con el almacén de datos y herramientas BI de tu preferencia, de forma limpia y segura.

## Seguridad y privacidad

**VTEX Data Pipeline** mejora la seguridad de los datos del sitio web de tu tienda online estableciendo una conexión directa y segura desde nuestro almacén de datos a tu herramienta de infraestructura de datos. Este enfoque elimina la necesidad de depender de procesos ETL (Extracción, Transformación y Carga) llevados a cabo por herramientas de terceros, proporcionando un control directo y seguro sobre el flujo de datos. Además, un aspecto clave de nuestra solución es la eliminación proactiva de toda la información personal identificable (PII) de los usuarios presentes en las tablas, lo que garantiza las mejores prácticas de protección de datos.  

## ¿Qué es un diccionario de datos?

En este diccionario tendrás acceso a todos los conjuntos de datos que ofrece **VTEX Data Pipeline**, cada uno detallado en una sección específica que describe sus características.

Cada sección proporciona información como:

- Características de los datos  
  - Origen  
  - Disponibilidad para el cliente  
  - Historial  
  - Intervalo mínimo de actualización  
- Tablas de conjunto de datos  
- Análisis con el conjunto de datos  
- Correlaciones con otros datos
