Cuando el usuario tiene una lista de compras (lista de bodas, lista de regalos), la barra de televentas antigua recuperaba los IDS de estas listas, que conducían directamente al área de gestión de esta. Este recurso ya no está disponible.

Todavía es posible acceder a las listas de compras desde el menú “Gestionar Lista”, que debe ponerse a disposición de los clientes en su sitio web a través de la URL con `/giftlist/manage`.
