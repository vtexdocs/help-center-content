Al [crear una configuración regional](https://help.vtex.com/es/docs/tutorials/configurar-configuraciones-regionales) para tu tienda, debes seleccionar un idioma y una región. La plataforma admite un conjunto predeterminado de idiomas y regiones que se pueden combinar para formar códigos de configuraciones regionales, como `es-MX` o `en-US`.

Las siguientes especificaciones técnicas se aplican a todas las configuraciones regionales admitidas:

* Codificación: UTF-8  
* Dirección del texto: de izquierda a derecha (LTR)  
* Alfabetos compatibles: latino, cirílico

A continuación se muestra la lista completa de idiomas y regiones compatibles.

## Idiomas admitidos

Puedes agregar configuraciones regionales utilizando los siguientes idiomas:

* Alemán (de)  
* Búlgaro (bg)  
* Checo (cs)  
* Croata (hr)  
* Danés (da)  
* Eslovaco (sk)  
* Esloveno (sl)  
* Español (es)  
* Estonio (et)  
* Finlandés (fi)  
* Francés (fr)  
* Griego (el)  
* Húngaro (hu)  
* Inglés (en)  
* Irlandés (ga)  
* Italiano (it)  
* Letón (lv)  
* Lituano (lt)  
* Maltés (mt)  
* Neerlandés (nl)  
* Polaco (pl)  
* Portugués (pt)  
* Rumano (ro)  
* Serbio (sr)  
* Sueco (sv)
