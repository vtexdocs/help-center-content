>ℹ️ Esta funcionalidad se encuentra en fase beta, lo que significa que estamos trabajando para optimizarla. Si desea migrar su tienda a la nueva página, ponte en contacto con <a href = "[https://support.vtex.com/hc/es/requests](https://support.vtex.com/hc/pt-br/requests)">soporte de VTEX</a>.

Para agregar un producto en el Admin VTEX, debes rellenar la información de los formularios de Productos y SKUs. Sigue los pasos a continuación para acceder a la nueva experiencia de agregar productos y SKUs:

1. En el Admin VTEX, accede a **Catálogo > ** **Productos y SKUs**.
2. Haz clic en `Agregar producto`.
3. Rellena los [campos del formulario de producto](https://docs.google.com/document/d/1gXmCDMgAVy4NHCm6JZfW2cXg683iIHWz/edit#heading=h.3whwml4).
4. Haz clic en `Guardar` o accede a la pestaña `SKUs`, si deseas [asociar SKUs](https://help.vtex.com/es/tutorial/cadastro-de-skus-beta--4ryZ6J45kwn3jDiQBxGiiN#) con este producto.

# Campos del formulario de producto

En esta página debes rellenar la información necesaria para agregar un producto. Consulta a continuación más detalles sobre los campos que deben completarse en cada sección del formulario.

## Información general

- **Nombre del producto:** nombre del producto a agregar. Utiliza vocabulario sencillo y evita términos en otro idioma o redacción compleja. Este campo es importante para SEO y se debe respetar el límite de 150 caracteres. Campo obligatorio.
- **Palabras sustitutas:** sinónimos de términos relacionados con el nombre de tu departamento o categoría. "Televisión", por ejemplo, puede sustituirse por "TV" o "Tele". Este campo es importante para ampliar el alcance de las búsquedas y debe respetarse el límite de 8.000 caracteres.
- **Marca:** ingresa o busca el nombre de la marca y luego añádela al registro. Para hacerlo, debes haber agregado la marca y que esté activa en tu tienda. Campo obligatorio.
- **Categoría:** selecciona la categoría a la que el producto pertenece. Para hacerlo, debes haber agregado la categoría y que esté activa en tu tienda. Campo obligatorio.
- **Código de referencia del producto:** rellena la combinación única de letras, números o ambos para identificación interna de la tienda.

## Descripción

- **Descripción:** información principal sobre el producto. Constituye un resumen sencillo y fácil de entender para el cliente.

## SEO

- **URL del producto:** utilizada para crear la URL del producto. Campo obligatorio. Para más información, consulta el artículo [¿Cómo definir la URL de un producto?](https://help.vtex.com/es/tutorial/como-a-url-de-produto-e-montada--frequentlyAskedQuestions_368).
- **Título de la página:** texto que se muestra en la pestaña del navegador y corresponde al título de la página del producto. Este campo es importante para SEO.
- **Metadescripción:** ingresa una breve descripción de la categoría. Es aconsejable no superar los 150 caracteres para que los motores de búsqueda puedan mostrarla correctamente en las páginas de resultados.

>ℹ️ Al rellenar la información de SEO, se mostrará una vista previa del contenido.

## Operaciones y logística

- **Políticas comerciales:** selecciona las [políticas comerciales](https://help.vtex.com/es/tutorial/como-funciona-uma-politica-comercial--6Xef8PZiFm40kg2STrMkMV#) aplicables al producto.
- **Código fiscal:** número de identificación tributaria del producto.

## Visibilidad

- **Mostrar en el sitio web:** selecciona si deseas que el producto esté visible en el sitio web. 
- **Mostrar cuando no tenga stock:** si activas esta opción, el producto agotado seguirá visible en tu tienda con la función [Avísame](https://help.vtex.com/es/tutorial/configurar-a-opcao-avise-me--2VqVifQuf6Co2KG048Yu6e#). El cliente podrá ingresar su email para recibir una notificación cuando el producto vuelva a estar disponible.
- **Prioridad en el orden de búsqueda:** valor numérico para definir la prioridad en la página de resultados de búsqueda. Para saber más, consulta [¿Cómo funciona el campo Score?](https://help.vtex.com/es/tutorial/como-funciona-o-campo-score--1BUZC0mBYEEIUgeQYAKcae).

## Atributos de categoría

Dependiendo de la categoría a la que pertenece el producto que creaste, en la sección **Atributos de categoría ** pueden mostrarse campos para rellenar. Estos campos son especificaciones que puedes añadir a tus SKU para atribuir características a los productos de tu tienda.

Consulta este tutorial para aprender a crear [las especificaciones de tu producto](https://help.vtex.com/es/tutorial/criando-um-campo-de-produto?locale=pt).

Si creaste algún campo de producto que sea obligatorio, el producto solo podrá activarse una vez que se haya rellenado la especificación.

>ℹ️ Si tras crear el producto no se muestra en el sitio web de tu tienda, consulta el artículo [¿Por qué el producto no aparece en el sitio web?](https://help.vtex.com/es/faq/por-que-o-produto-nao-aparece-no-site--frequentlyAskedQuestions_382) para saber cómo solucionar este problema.
