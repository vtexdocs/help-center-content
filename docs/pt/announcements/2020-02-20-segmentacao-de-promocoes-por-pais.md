A partir do dia 21 de Fevereiro de 2020, a VTEX apresenta uma nova feature para clientes que têm operação em vários países: o filtro de promoção por país.

Essa funcionalidade permite que o usuário segmente o uso de uma promoção para os países que recebem entregas da sua loja.

Conferido isso, as localidades serão apresentadas no Admin da loja no módulo Promoções.

Para saber mais detalhes sobre Promoções, veja nossa [track](https://help.vtex.com/pt/tracks/promocoes--6asfF1vFYiZgTQtOzwJchR/2a2D0K85Ahvs4hLnL3Ag7N).
