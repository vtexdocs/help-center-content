As labels de ação usam até três palavras para indicar uma ação que o usuário deseja ou precisa executar. Esse padrão é usado com frequência em componentes como botões ou menus.

## Geral

Mantenha o texto o mais curto possível, mas suficientemente descritivo.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| Adicionar produto | Adicionar novo produto ao catálogo |

Use um verbo no infinitivo seguido de um objeto.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>Criar produto</li><li>Visitar loja</li></ul> | <ul><li>Novo produto</li><li>Loja</li></ul> |

Use as mesmas labels para ações que se repetem. As labels mais comuns estão listadas no glossário.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>Adicionar produto</li><li>Ver detalhes</li><li>Editar</li><li>Renomear</li></ul> | <ul><li>Inserir produto</li><li>Ver detalhes</li><li>Modificar</li><li>Alterar nome</li></ul> |

Use letra maiúscula somente na primeira letra da frase e em nomes próprios.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| Testar nova experiência | Testar Nova Experiência |

Não use pontuação ou pronomes pessoais e evite usar artigos.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>Adicionar produto</li><li>Pré-visualizar conteúdo</li></ul> | <ul><li>Adicionar um produto</li><li>Pré-visualizar meu conteúdo</li></ul> |

## Específico

Use labels precisas que reflitam a ação exata.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>Publicar</li><li>Renomear</li><li>Saiba mais</li></ul> | <ul><li>Confirmar</li><li>Editar</li><li>Clique aqui</li></ul> |

Se o contexto já deixa claro a que objeto se refere a ação, não o repita na label.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>Label de conteúdo: Produto</li><li>Label de ação: Editar</li></ul><ul><li>Label de conteúdo: Pedido</li><li>Label de ação: Faturar</li></ul> | <ul><li>Label de conteúdo: Produto</li><li>Label de ação: Editar produto</li></ul><ul><li>Label de conteúdo: Pedido</li><li>Label de ação: Faturar pedido</li></ul> |

Quando um link está no meio de um parágrafo, não é necessário iniciar com maiúscula ou verbo, a menos que seja o título exato de um artigo.
