Cadastrar um novo vendedor é simples. Siga o passo a passo abaixo:

## Como cadastrar um vendedor
<ol>
  <li>No menu lateral, acesse **Cadastro &gt; Vendedores**.</li>
  <li>Clique em **Cadastrar Vendedor**.</li>
  <li>Preencha as seguintes informações obrigatórias:
    <ul>
      <li>Foto do vendedor</li>
      <li>CPF</li>
      <li>Nome completo</li>
      <li>Nome para exibição no sistema</li>
      <li>Sexo</li>
      <li>Data de nascimento</li>
      <li>Tipo (tipo de contratação: fixo ou temporário)</li>
      <li>Data de início</li>
      <li>Data de término (do contrato de experiência ou do período temporário)</li>
      <li>Senha do vendedor no tablet</li>
      <li>Código do vendedor (código que identifica o vendedor no sistema de PDV da loja)</li>
    </ul>
  </li>
  <li>Clique em **Cadastrar Vendedor** para **salvar** o cadastro.</li>
</ol>
