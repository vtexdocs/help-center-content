É possível acompanhar o feedback registrado pelo gestor diretamente na plataforma Indeva.

<ol>
  <li>No menu lateral, acesse **Feedback &gt; Gerente &gt; Visualizar**.</li>
  <li>Consulte os indicadores exibidos e o feedback registrado.</li>
  <li>Se necessário, adicione comentários para alinhar um plano de ação com o gestor.</li>
</ol>
