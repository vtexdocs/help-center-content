Na plataforma Indeva, é possível acompanhar o feedback registrado pelo gestor.

Para acessar:

1. No menu, selecione **Feedback > Gerente > Visualizar**.
2. Consulte os indicadores exibidos e o feedback recebido.
3. Caso necessário, adicione comentários para alinhar um plano de ação com o gestor.

![](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/pt/tutorials/indeva-by-vtex/gerentes/visualizando-feedback-indeva_1.png)

> ℹ️ Use este recurso para alinhar estratégias da loja com o gestor e planejar ações de melhoria.
