Para transferir um vendedor de uma loja para outra seja de forma temporária ou definitiva é necessário encerrar o vínculo anterior antes de criar o novo.

Abaixo estão as etapas para realizar essa operação corretamente:

- Para desligar o colaborador da loja de origem, acesse: **Cadastros > Vendedores > Ações > Férias ou Inativar > Desligamento**

> ⚠️ A data de início na nova loja deve ser posterior à data de desligamento da loja anterior.<br><br> **Exemplo:** se o cadastro foi encerrado em 01/01/2024, o novo vínculo deve começar em 02/01/2024.

- Para cadastrar o colaborador na nova loja, acesse: **Cadastros > Vendedores > Cadastrar vendedor**

Preencha as informações obrigatórias e defina uma **data de início posterior** à data de desligamento registrada anteriormente.
