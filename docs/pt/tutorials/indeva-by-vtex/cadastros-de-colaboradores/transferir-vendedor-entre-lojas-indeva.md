<section>
  <h1>Transferir vendedor entre lojas na plataforma Indeva</h1>

  
    É possível transferir um vendedor de uma loja para outra, seja de forma temporária ou definitiva. 
    Para isso, é necessário **encerrar o vínculo anterior** antes de criar o novo.
  

  <h2>Realizar transferência de vendedor</h2>

  Siga os passos abaixo para realizar essa operação corretamente:

  <ol>
    <li>
      **Desligar o colaborador da loja de origem:**<br>
      Acesse **Cadastros &gt; Vendedores &gt; Ações &gt; Férias ou Inativar &gt; Desligamento**.
    </li>
  </ol>

  > ⚠️ A data de início na nova loja deve ser **posterior** à data de desligamento da loja anterior.

  <ol start="2">
    <li>
      **Cadastrar o colaborador na nova loja:**<br>
      Acesse **Cadastros &gt; Vendedores &gt; Cadastrar vendedor**.
    </li>
  </ol>

  
    Preencha as informações obrigatórias e defina uma 
    **data de início posterior** à data de desligamento registrada anteriormente.
