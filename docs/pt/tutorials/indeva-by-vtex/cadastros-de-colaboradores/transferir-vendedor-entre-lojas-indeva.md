É possível transferir um vendedor de uma loja para outra, seja de forma temporária ou definitiva. Para isso, é necessário encerrar o vínculo anterior antes de criar o novo.

Siga os passos abaixo para realizar essa operação corretamente:

<ol>
  <li>
    Para desligar o colaborador da loja de origem, acesse:<br>
    **Cadastros &gt; Vendedores &gt; Ações &gt; Férias ou Inativar &gt; Desligamento**
  </li>
</ol>

> ⚠️ A data de início na nova loja deve ser **posterior** à data de desligamento da loja anterior.

<ol start="2">
  <li>
    Para cadastrar o colaborador na nova loja, acesse:<br>
    **Cadastros &gt; Vendedores &gt; Cadastrar Vendedor**
  </li>
</ol>

Preencha as informações obrigatórias e defina uma **data de início posterior** à data de desligamento registrada anteriormente.
