Na plataforma Indeva, só é possível cadastrar uma meta por ciclo.  
A próxima meta só pode começar após o término da anterior.  
Se houver sobreposição de datas, o sistema exibirá uma mensagem de conflito.


Para resolver esse tipo de conflito, siga os passos abaixo:

<ul>
  <li>
    **Verifique se já existe uma meta para o ciclo atual.**<br />
    Se houver uma meta com a mesma data de início e término, edite a meta existente em **Ações &gt; Editar**.
  </li>

  <li>
    **Garanta que a nova meta comece após o término da anterior.**<br />
    Se houver sobreposição de datas, ajuste a **data de início** da nova meta para o dia seguinte ao término da anterior.
    <br /><br />
    *Exemplo:* se a primeira meta vai de **01/07/2022 a 31/07/2022**, a próxima deve começar em **01/08/2022**.
  </li>

  <li>
    **Verifique o mês de referência da meta.**<br />
    Se já existir uma meta cadastrada com o mesmo mês de referência, a nova meta deve ser registrada com o mês seguinte.
    <br /><br />
    *Exemplo:* se a meta atual tem mês de referência **07** (julho), a próxima deve usar **08** (agosto).
  </li>
</ul>

> ⚠️ Quando o ciclo mensal inclui dois meses diferentes, defina como mês de referência aquele que tiver mais dias dentro do ciclo.
