Para que as metas sejam calculadas corretamente, o ciclo considera apenas dias úteis. Por isso, é necessário indicar no calendário os dias não úteis (feriados e folgas) do ciclo para um vendedor específico ou para toda a equipe.

Para saber mais como cadastrar, acesse o artigo [Folgas e feriados](/pt/tutorial/folgas-e-feriados-indeva--1EowcHY0U9nP2oov3mN26S).
