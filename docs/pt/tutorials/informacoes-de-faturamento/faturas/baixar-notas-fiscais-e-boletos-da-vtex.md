No Admin VTEX, você pode acessar diferentes documentos relacionados ao faturamento, como notas fiscais e, para clientes com faturamento no Brasil, boletos. 

Este artigo explica como localizar e baixar esses documentos, caso você não os tenha recebido por e-mail.

## Antes de começar

Para acessar esta seção do Admin VTEX, é necessário ter um perfil de acesso financeiro. Para saber como configurar esse perfil, consulte o artigo [Criar um perfil de acesso financeiro](/pt/docs/tutorials/criar-um-perfil-de-acesso-financeiro) ou peça ao proprietário da loja que ajuste suas permissões.

## Instruções

### Baixar uma nota fiscal

Para baixar uma nota fiscal, siga estes passos:

1. No Admin VTEX, clique no seu avatar — identificado pela inicial do seu e-mail — no canto superior direito.
2. Clique em **Informações de faturamento > Faturas**.
3. Localize a fatura desejada e clique em **Abrir nota fiscal**.
4. A nota fiscal abrirá em uma nova aba. Para salvar o arquivo em PDF, use as opções de baixar ou de imprimir disponíveis no navegador.

### Baixar um boleto

>⚠️ As instruções desta seção se aplicam apenas a clientes faturados no Brasil.

Clientes com faturamento no Brasil podem emitir uma nova via do boleto seguindo estas etapas:

1. No Admin VTEX, clique no seu avatar — identificado pela inicial do seu e-mail — no canto superior direito.
2. Clique em **Informações de faturamento > Faturas**.
3. Localize o boleto desejado e clique em **Efetuar pagamento**.
4. O boleto abrirá em uma nova aba, com código de barras e QR Code para pagamento via Pix. Para salvar ou imprimir o arquivo, clique em `Clique aqui para imprimir`.
