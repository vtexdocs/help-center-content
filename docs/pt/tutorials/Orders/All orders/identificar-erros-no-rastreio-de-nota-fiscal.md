Para se certificar do funcionamento do rastreio de nota fiscal, analise os itens abaixo. Estes são os principais motivos para o não funcionamento do rastreio:

- Verifique se o número de rastreio está errado;
- Verifique se a URL de rastreio está errada;
- Verifique se a transportadora está errada;
- Acesse a URL de rastreio passando o número no navegador e verifique se acessa a página de acompanhamento de rastreio;
- Verifique se a transportadora está na [lista de transportadoras cadastradas para o rastreio](/pt/faq/quais-transportadoras-disponibilizam-o-rastreio-de-frete "lista de transportadoras cadastradas para o rastreio").

_Ajuste esses itens e acompanhe o disparo dos e-mails pela linha do tempo no pedido._
