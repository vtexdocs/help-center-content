>⚠️ A atualização agendada pode demorar até 30 minutos para ser efetivada e aparecer na sua página.

Depois de [criar uma nova versão de conteúdo](https://help.vtex.com/pt/tutorial/gerenciando-versoes-de-conteudo--4loXo98CZncY0NnjKrScbG?&utm_source=autocomplete), você pode fazer um agendamento para publicá-la na sua loja por meio do recurso **Visibilidade**.

O recurso Visibilidade permite que você ative imediatamente a nova versão do conteúdo na sua loja, programe quando publicar o conteúdo ou quando desativar a nova versão. Você também pode escolher onde deseja aplicar a nova versão: em uma rota (**esta URL**) ou em rotas que compartilham o mesmo template (**este template**).

Usaremos o bloco [Slider-Layout](https://developers.vtex.com/vtex-developer-docs/docs/vtex-slider-layout) de exemplo para ativar uma nova versão de conteúdo para sua loja.

1. No Admin VTEX, acesse **Storefront > Site Editor**.

2. Clique no bloco desejado (exemplo, Slider-Layout) e selecione **VERSÕES.**

3. Clique no conteúdo que deseja publicar.

4. Na seção **Visibilidade,** selecione quando o conteúdo será ativado: 

| Função     | Descrição  |
| ---------- | ---------- |
| **Ative agora** | O conteúdo será ativado imediatamente. |
| **Definir uma data de início** | Programe a data e a hora para ativar o conteúdo. |
| **Definir uma data de fim**  | Programe a data e a hora para desativar o conteúdo. Assim que o conteúdo for desativado, a última versão o substituirá. |

<div style="background-color:#FCF8F2; border-left: 2px solid #F0AD4E; border-top-left-radius: 2px; border-bottom-left-radius: 2px; padding: 15px; margin-bottom: 10px">
Se você não selecionar as opções da seção Visibilidade, você apenas terá uma nova versão salva, mas esta não será publicada em sua vitrine.
</div>
5. Depois de selecionar quando o conteúdo estará disponível, escolha onde deseja que seja aplicado:

<ul>
  <li><b>esta URL:</b> O conteúdo está disponível apenas para uma rota de sua escolha.</li>
  <li><b>este template:</b> O conteúdo está disponível para todas as rotas com o mesmo template.</li>
</ul>
6. Uma vez que você configurou a seção de visibilidade, clique em `SALVAR`.
