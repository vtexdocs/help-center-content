No app **Indeva – Lista da Vez**, o vendedor pode acessar os feedbacks registrados para ele, incluindo elogios e orientações com base nos resultados de desempenho.  

## Como visualizar

1. Abra o app **Indeva – Lista da Vez**.  
2. Clique na aba **Metas**.  
3. Selecione o nome do vendedor.  
4. Digite a senha para acessar as informações.
