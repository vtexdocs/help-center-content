O **Espaço de trabalho** é a seção de configurações do projeto no VTEX CX Platform. Nele, você pode definir as preferências gerais do projeto, gerenciar credenciais utilizadas pelos agentes e consultar o histórico de alterações realizadas.

## Acessar o Espaço de trabalho

Para acessar as configurações do seu projeto, siga as etapas abaixo:

1. Na página inicial, localize o projeto desejado.
2. Clique no menu `⋮` ao lado do nome do projeto e selecione **Configurações**.
3. No menu lateral, em **Configurações**, clique em **Espaço de trabalho**.

## Preferências do Projeto

A aba **Preferências do Projeto** permite configurar as informações gerais do projeto.

Para atualizar as preferências, siga as etapas abaixo:

1. No campo **Nome do projeto**, insira ou edite o nome do projeto.
2. No campo **Descrição do projeto**, insira uma descrição. Este campo é obrigatório.
3. No campo **Fuso horário**, selecione o fuso horário correspondente à operação da sua loja.
4. No campo **Idioma**, selecione o idioma do projeto.
5. Clique em `Salvar alterações`.

## Credenciais

A aba **Credenciais** exibe as credenciais utilizadas pelos agentes do projeto para se conectar a serviços externos.

As credenciais estão organizadas em duas seções:

- **Credenciais usadas pelos agentes oficiais:** credenciais associadas aos agentes nativos da plataforma.
- **Credenciais usadas pelos agentes personalizados:** credenciais associadas aos agentes criados pela sua equipe.

> ℹ️ Caso nenhuma credencial tenha sido configurada, a mensagem *Nenhuma credencial informada* será exibida em cada seção.

## Histórico de Alterações

A aba **Histórico de Alterações** registra todas as modificações realizadas no projeto pelo Agent Builder.

Nessa aba, você pode:

- Visualizar a lista de alterações com o autor, a descrição da ação e a data de cada modificação.
- Filtrar as alterações pelo tipo usando o menu suspenso **Todas as alterações** no canto superior direito.

### Detalhes do projeto

Para consultar informações técnicas sobre o backend e os modelos utilizados no projeto, clique em **Detalhes do Projeto** no canto superior direito da tela.

O modal exibe as seguintes informações:

| Campo | Descrição |
| --- | --- |
| **Backend dos agentes** | Provedor de IA utilizado pelos agentes do projeto. |
| **Modelo do Manager** | Versão do modelo de linguagem do agente gerenciador. |
| **Modelos dos agentes** | Versões dos modelos de linguagem de cada agente configurado no projeto. |
