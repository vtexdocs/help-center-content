A Bit.One é um [gateway](/pt/tutorial/o-que-e-um-gateway-de-pagamentos) especializado no processamento de pagamentos usando bitcoins.

Para configurar a Bit.One, siga os passos abaixo:

## Configurar afiliação de gateway Bit.One
1. Acesse o módulo __Pagamentos__.
2. Clique em __Configurações__.
3. Na aba __Afiliações de Gateways__, clique no botão __+__.
4. Clique no conector __Bitone__.
5. No campo __Application Key__, informe a chave da aplicação, fornecida pela Bit.One.
7. Em __Application Token__, preencha o campo com o token fornecido pela Bit.One.
8. Clique em __Salvar__.

## Configurar condição de pagamento
Depois de seguir os passos indicados, a Bit.One vai estar configurada na sua loja. Assim, você vai poder usá-la para processar pagamentos usando bitcoins. Para saber como cadastrar pagamentos com a criptomoeda, leia [este artigo do Help](/pt/tutorial/como-habilitar-pagamentos-com-bitcoin).
