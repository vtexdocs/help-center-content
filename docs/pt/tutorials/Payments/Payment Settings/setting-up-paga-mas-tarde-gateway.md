Na plataforma VTEX, você pode configurar o [gateway](/pt/tutorial/o-que-e-um-gateway-de-pagamentos) Paga+Tarde para receber pagamentos por meio de promissórias.

Para configurar o Paga+Tarde, siga os passos abaixo:

## Configurar afiliação de gateway Paga+Tarde
1. Acesse o módulo __Pagamentos__.
2. Clique em __Configurações__.
3. Na aba __Afiliações de Gateways__, clique no botão __+__.
4. Clique no conector __PagaMasTarde__.
5. No campo __Application Key__, informe a chave da aplicação, fornecida pelo Paga+Tarde.
7. Em __Application Token__, preencha o campo com o token fornecido pelo Paga+Tarde.
8. Clique em __Salvar__.

## Configurar condição de pagamento
Depois de seguir os passos indicados, o PagaMasTarde vai estar configurado na sua loja. Assim, quando você for cadastrar uma promissória, ele vai estar disponível no campo __Processar com a afiliação__. 
Para saber como configurar pagamentos customizados (como as promissórias), acesse [este artigo do Help](/pt/tutorial/como-configurar-pagamento-customizado).
