O PlaceToPay é um gateway que oferece soluções transparentes para lojas de grande e pequeno porte. Entre seus diferenciais está a fácil integração com os diversos sistemas que integram o fluxo de pagamento.

Para configurar o PlaceToPay, siga os passos abaixo:

## Configurar afiliação de gateway PlaceToPay
1. Acesse o módulo __Pagamentos__.
2. Clique em __Configurações__.
3. Na aba __Afiliações de Gateways__, clique no botão __+__.
4. Clique no conector __PlaceToPay__.
5. Preencha os campos de cadastro com os dados fornecidos pelo PlaceToPay (campos _API Login_ e _API Key_).
6. No campo __Forma de Autorização__, escolha se quer que os pagamentos passem por uma pré-autorização ou não.
7. Em __Moeda__, defina a moeda em que a transação vai acontecer.
8. Em __Linguagem__, escolha um idioma.
9. No campo __E-mail ao Lojista__, selecione se deseja ou não que o sistema envie um e-mail para você sempre que uma compra for realizada.
10. Já no campo __E-mail ao Cliente__, selecione se deseja ou não que o sistema envie um e-mail ao seu cliente quando ele fizer uma compra.
11. Clique em __Salvar__.

## Configurar condição de pagamento
Depois de seguir os passos indicados, o PlaceToPay vai estar configurado na sua loja. Assim, quando você for cadastrar uma condição de pagamento, ele vai estar disponível no campo __Processar com a afiliação__ (desde que o meio de pagamento seja compatível com o PlaceToPay). Para saber como definir condições de pagamento, acesse [este artigo do Help](/pt/tutorial/condicoes-de-pagamento).
