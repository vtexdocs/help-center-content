Na VTEX, é possível realizar a integração com o provedor de pagamentos Gateway 4all. Com este conector, sua loja pode realizar vendas por meio de cartões de crédito.

Para configurar a afiliação Gateway 4all, acesse [Cadastrar afiliações de gateway](https://help.vtex.com/pt/tutorial/afiliacoes-de-gateway--tutorials_444#).

Para configurar as condições de pagamento, acesse [Configurar condições de pagamento](https://help.vtex.com/pt/tutorial/condicoes-de-pagamento#).

Depois de seguir os passos indicados, o conector Gateway 4all pode demorar até 10 minutos para aparecer no checkout da sua loja como uma opção de pagamento.
