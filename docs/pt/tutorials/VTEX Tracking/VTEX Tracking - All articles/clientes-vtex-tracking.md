Clientes são os consumidores finais de uma loja que se beneficiam do serviço VTEX Tracking para rastreio de pedidos. Podem ser tanto pessoas físicas, para negócios B2C, quanto pessoas jurídicas, para negócios B2B, ou rastreio de pedidos de  fornecedores. Neste artigo você aprenderá a: 

- Consultar clientes  
- Criar novo cliente  
- Editar dados de um cliente  
- Campos de cadastro de cliente  

## Consultar clientes

1. Acesse o módulo **Configurações da Conta**.  
2. Acesse a aba **Clientes**.  
3. Clique em **Consultar Clientes**.  

Uma vez no painel, é possível:

- Buscar clientes pelo nome, sobrenome, CPF, Razão Social, CNPJ ou e-mail.  
- Filtrar a busca por clientes, selecionando a opção `Todos`, `Pessoa Física` ou `Pessoa Jurídica`.  
- Selecionar um cliente, e conferir detalhes de sua conta, clicando no botão`:`.  
- Excluir  um cliente, clicando no botão `:` e selecionando a opção `Excluir`.  

## Criar novo cliente

<div class=”alert alert-info”>
Cada vez que é lançada uma entrega para um cliente não cadastrado, o sistema registra seu perfil no painel <b>Consultar clientes</b>, como se fosse um cliente novo. Para evitar essa repetição, crie um perfil manualmente para seus clientes, e vincule um Código Externo a cada um. Confira os Campos de Cadastro para saber mais sobre Código Externo.
</div>

1. Acesse o módulo **Configurações da Conta**.  
2. Acesse a aba **Clientes**.  
3. Clique em **Novo Cliente**.  
4. Preencha os campos de cadastro de cliente.   
5. Clique em **Criar**.  

## Editar dados de um cliente

1. Acesse o módulo **Configurações da Conta**.  
2. Acesse a aba **Cliente**.  
3. Clique em **Consultar Clientes**.  
4. Clique no botão `:` ao lado do nome do usuário.  
5. Altere os campos desejados.  
6. Clique em **Salvar**.  

### Campos de cadastro de cliente

Cada vez que é lançada uma entrega para um cliente, o sistema já preenche os campos descritos abaixo. Você também pode preenchê-los manualmente para criar um novo cliente. 

#### Dados do Cliente

- **Tipo:** (obrigatório) selecione entre as opções Pessoa Física ou Pessoa Jurídica.   
- **Nome:** (obrigatório) nome do cliente.  
- **Sobrenome:** (obrigatório) sobrenome do cliente.  
- **CPF:**  número do CPF do cliente, pode ser inserido para fins de consulta.  
- **Telefone:**  número de contato de um cliente, pode ser inserido para fins de consulta.   
- **E-mail:** (obrigatório) e-mail do usuário para realizar o login na conta do portador.  
- **Código externo:** atribua um código de sua escolha, para a identificação do cliente. Pode ser o código já existente para aquele cliente no CRM da sua loja, por exemplo. Cada vez que é lançada uma entrega para um cliente não cadastrado, o sistema registra seu perfil no painel *Consultar Clientes*, como se fosse um cliente novo. Para evitar essa repetição, crie um perfil manualmente para seus clientes, e vincule um Código Externo a cada um.   

#### Endereço

- **CEP:** CEP do endereço cadastrado para esse cliente.   
- **Endereço:**  endereço do cliente.  
- **Número:** número da casa, apartamento, ou lote do endereço acima.  
- **Complemento:** (opcional) complemento do endereço cadastrado.  
- **Bairro:** bairro correspondente ao endereço cadastrado.  
- **Cidade:** cidade correspondente ao endereço cadastrado.  
- **Estado:** selecione o estado correspondente ao endereço, entre as opções já disponíveis no campo.  
- **País:** país correspondente ao endereço cadastrado.
