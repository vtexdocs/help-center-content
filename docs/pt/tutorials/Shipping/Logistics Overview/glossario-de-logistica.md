O glossário abaixo contém definições dos termos relacionados ao módulo **Envio.**

| Termo | Significado | Saiba mais |
| ----------- | ----------- |----------- |
| <b>Adicional de frete</b> | Custos extras que podem ser adicionados ao custo para o envio (<a href="https://help.vtex.com/pt/tutorial/tarifas-de-envio--1Balpg3rv0854udEPedvMM">tarifas de envio</a>). Os adicionais possíveis são: <ul><li><a href="https://help.vtex.com/pt/tutorial/adicionais-de-frete--2vqGwMn0LabkOHY6zSHYNV#adicional-por-peso">Adicional por peso</a></li> <li><a href="https://help.vtex.com/pt/tutorial/adicionais-de-frete--2vqGwMn0LabkOHY6zSHYNV#adicional-por-preco">Adicional por preço</a></li> <li><a href="https://help.vtex.com/pt/tutorial/adicionais-de-frete--2vqGwMn0LabkOHY6zSHYNV#adicional-de-seguro">Adicional de seguro</a></li> <li><a href="https://help.vtex.com/pt/tutorial/adicionais-de-frete--2vqGwMn0LabkOHY6zSHYNV#adicional-de-frete">Adicional de frete</a></li></ul> | <a href="https://help.vtex.com/pt/tutorial/adicionais-de-frete--2vqGwMn0LabkOHY6zSHYNV">Adicionais de frete</a> | 
| <b>Canais de entrega</b> | Tipos de envio: entrega e ponto de retirada. A entrega é o envio que o cliente recebe no endereço desejado, geralmente com intermédio de uma transportadora. O ponto de retirada é o local físico que a loja disponibiliza para que o cliente retire seus pedidos. |  |
| <b>Capacidade de entrega</b> | Limite de entregas que podem ser realizadas em um período.| <a href="https://help.vtex.com/pt/tutorial/entrega-agendada--22g3HAVCGLFiU7xugShOBi#capacidade-de-entrega">Capacidade de entrega</a> |
| <b>Custo final do envio</b> | Valor final do envio, considerando os custos fixos, valores adicionais e custos de doca. | <a href="https://help.vtex.com/pt/tutorial/custo-final-do-envio--5bwhIO108VA5Y2YOpef9lV">Custo final do envio</a> |
| <b>Divisão de carga/Divisão de pacote</b> | Funcionalidade que permite dividir a entrega do carrinho de compras quando o cálculo total do peso ultrapassa o limite aceito pelas transportadoras. | <ul><li><a href="https://help.vtex.com/pt/tutorial/como-funciona-a-divisao-de-carga--tutorials_109">Divisão de carga</a></li> <li><a href="https://help.vtex.com/pt/tutorial/divisao-de-pedidos-e-divisao-de-entregas--jQvzA6QgSd51e2p6bthoV">Divisão de pedidos e divisão de entregas</a></li> |
| <b>Doca</b> | Entidade que conecta o estoque e a política de envio. | <a href="https://help.vtex.com/pt/tutorial/doca--5DY8xHEjOLYDVL41Urd5qj">Doca</a> |
  | <b> Entrega</b> | Tipo de envio em que o cliente recebe seus pedidos no seu endereço. |   |
  | <b>Envio</b> | Despacho dos pedidos. O envio pode ser por <b>entrega</b> (no endereço do cliente) ou <b>retirada</b> (retirada pelo cliente em um determinado local). | |
  | <b>Estoque</b> | Identificação do local de armazenamento das mercadorias da loja. | <a href="https://help.vtex.com/pt/tutorial/estoque--6oIxvsVDTtGpO7y6zwhGpb">Estoque</a> |
  | <b> Estoque crítico</b> | Nome para o alerta de quantidade baixa de itens em estoque. | <a href="https://help.vtex.com/pt/tutorial/configurar-o-alerta-de-estoque-critico--6FD0GHeQPCsKIMgkQ88SGu">Estoque crítico</a> |
  | <b>Fator cúbico de peso</b> | É a relação entre o peso e o volume de um pacote. Esse fator auxilia na estimativa do espaço ocupado e a massa dos pacotes na entrega. | <a href="https://help.vtex.com/pt/tutorial/como-o-peso-cubado-e-calculado--tutorials_128">Fator cúbico de peso</a> |
    | <b>Inventário</b> | Relação dos SKUs que estão no estoque e suas respectivas quantidades. | <a href="https://help.vtex.com/pt/tutorial/gerenciar-itens-em-estoque--tutorials_139">Inventário</a> |
     | <b>Marketplace</b> | Loja responsável pela venda dos itens, mas não pelo envio. | <a href="https://help.vtex.com/pt/tutorial/o-que-e-um-marketplace--680lLJTnmEAmekcC0MIea8">Marketplace</a> | 
       | <b>Modal</b> | Classificação dos tipos de itens que são entregues pelas transportadoras. | <a href="https://help.vtex.com/pt/tutorial/como-funciona-o-modal--tutorials_125">Modal</a> | 
 | <b>Pacote</b> | Conjunto de itens de um pedido que são agrupados. Um único pedido pode ter mais um pacote. | <a href="https://help.vtex.com/pt/tutorial/divisao-de-pedidos-e-divisao-de-entregas--jQvzA6QgSd51e2p6bthoV">Divisão de pedidos e divisão de entregas</a> | 
 | <b>Planilha de frete</b> | A planilha de frete é uma maneira de cadastrar os preços de envio que serão oferecidos aos seus clientes por uma determinada transportadora. | <a href="https://help.vtex.com/pt/tutorial/planilha-de-frete--tutorials_127">Planilha de frete</a> | 
 | <b>Política comercial</b> | Conjunto de configurações comerciais que a VTEX oferece para as lojas ajustarem suas estratégias de negócios de acordo com moeda, país, envios, preços, clientes, catálogos e pagamento. | <a href="https://help.vtex.com/pt/tutorial/como-funciona-uma-politica-comercial--6Xef8PZiFm40kg2STrMkMV">Política comercial</a> | 
 | <b>Política de envio</b> | Descreve preços e prazos acordados pela loja com uma dada transportadora. | <a href="https://help.vtex.com/pt/tutorial/politica-de-envio--tutorials_140">Política de envio</a> |
 | <b>Ponto de retirada (Pick up points)</b> | Locais físicos para retirada de pedidos pelos clientes. | <a href="https://help.vtex.com/pt/tutorial/como-funcionam-pontos-de-retirada--2fljn6wLjn8M4lJHA6HP3R">Ponto de retirada</a> |
 | <b>Remessa/Agrupamento de pacotes</b> | Configuração de agrupamento de pacotes para envio, a fim de minimizar o preparo e a retirada dos pacotes. Por exemplo, se a remessa é configurada para cinco dias, todos os pedidos com preparo de até cinco dias serão agrupados e enviados juntos. | <a href="https://help.vtex.com/pt/tutorial/como-funciona-a-remessa--tutorials_118">Remessa</a> |
| <b>Seller</b> | Lojas responsáveis pelo envio dos pedidos, mas não necessariamente pela venda.  | <a href="https://help.vtex.com/pt/tutorial/o-que-e-um-seller--5FkLvhZ3Few4CWWIuYOK2w">Seller</a> |
 | <b>Sellers white label</b> | Seller que não é especificado para o cliente final. Ao comprar de um seller white label, o cliente não sabe qual loja é responsável pelo envio do produto e tem contato apenas com o marketplace. Geralmente utilizado para contas franquia.  | <a href="https://help.vtex.com/pt/tutorial/algoritmo-de-selecao-de-sellers-white-label--3MemNQ4pKkWCpMdzI27AHa">Seleção de sellers white label</a> |
| <b>Simulador de envío</b> | Ferramenta para simulação de envio. A simulação permite a análise de disponibilidade dos SKUs, das transportadoras e dos pontos de retirada.  | <a href="https://help.vtex.com/pt/tutorial/simulador-de-envio--tutorials_144">Simulador de envio</a> || <b>SKU</b> | Sigla em inglês para<i>stock</i>Keeping Unit. É a unidade básica de um produto do catálogo, com a descrição mais básica de um item (por exemplo, o produto Camisa tem o SKU Camisa azul tamanho M). | <a href="https://help.vtex.com/pt/tutorial/o-que-e-um-sku--1K75s4RXAQyOuGUYKMM68u">SKU</a> |
| <b>SLA</b> | É a sigla para Service Level Agreement, que significa acordo de nível de serviço. Essa sigla tem aplicação em diferentes contextos. Na logística da VTEX, significa o acordo entre a loja e o cliente final quanto à estimativa do tempo de entrega. A plataforma VTEX calcula o tempo estimado de entrega de um pedido e apresenta para o cliente final no checkout. Essa estimativa de entrega é o SLA da loja com o cliente final. O cálculo dessa estimativa leva em consideração as  configurações das políticas de envio e das docas. |<a href="https://developers.vtex.com/vtex-rest-api/reference/calculatesla">SLA</a>|
  | <b>Tarifa de doca</b> | <b>Tarifa adicional</b> relacionada ao manuseio dos produtos que saem do estoque e vão para a doca. Esse valor é configurado no <a href="https://help.vtex.com/pt/tutorial/gerenciar-estoque--tutorials_137">Estoque</a>, na seção <b>Relação com docas.</b> | <a href="https://help.vtex.com/pt/tutorial/gerenciar-estoque--tutorials_137#campos-de-cadastro">Campos de cadastro de estoque</a> |
  | <b>Tarifa fixa (envio)</b> | Tarifa base da Tarifa de envio.  | <a href="https://help.vtex.com/pt/tutorial/tarifas-de-envio--1Balpg3rv0854udEPedvMM">Tarifas de envio</a> |
  | <b>Tarifas de envio</b> | São os custos configurados nas Políticas de envio. A tarifa de envio é composta pela Tarifa fixa e pelos adicionais de frete. | <a href="https://help.vtex.com/pt/tutorial/tarifas-de-envio--1Balpg3rv0854udEPedvMM">Tarifas de envio</a> |
