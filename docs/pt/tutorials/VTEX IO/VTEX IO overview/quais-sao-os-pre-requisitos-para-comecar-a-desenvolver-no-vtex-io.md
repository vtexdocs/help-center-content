Antes de começar a desenvolver no VTEX IO, verifique se você já tem:

- npm instalado
- Node.js instalado
- permissão de acesso a alguma conta VTEX

Também é recomendável o conhecimento de algumas tecnologias, como:

## Front-end

- React
- GraphQL
 
## Back-end

- TypeScript
- Node

## Leitura recomendada

React:

  - Introdução: https://reactjs.org/docs/hello-world.html
  - Aprofundamento: https://reactjs.org/tutorial/tutorial.html

GraphQL:
  - Introdução: http://graphql.org/learn/
  - Aprofundamento: https://www.howtographql.com/basics/0-introduction/

Apollo (framework que ajuda o React e o GraphQL a trabalharem juntos):
  - https://www.apollographql.com/docs/react/recipes/simple-example.html

Fullstack (React + Apollo + GraphQL):
  - https://www.howtographql.com/react-apollo/0-introduction/


Inicialmente, é possível apenas o desenvolvimento front-end. Portanto, o escopo de conhecimento inicial necessário limita-se ao React.
