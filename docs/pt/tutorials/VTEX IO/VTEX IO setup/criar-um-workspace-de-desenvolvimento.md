Para começar a desenvolver, você deveria criar um workspace de desenvolvimento. Para isso, basta digitar:

`vtex workspace create {name}`

- `{name}` deve ser substituído pelo nome que você deseja dar ao workspace.

Você também pode simplesmente usar o comando para mudar para um novo workspace: `vtex use {name}`. Ao identificar que ainda não existem workspaces novos, o sistema perguntará se você deseja criar um. Ao confirmar, você será levado automaticamente para o novo espaço de trabalho.
