Este artigo tem como objetivo enunciar algumas boas práticas em relação ao uso das imagens dos seus produtos. Para obter bons resultados com as suas imagens e solucionar possíveis problemas de importação, siga as sugestões abaixo:

- O nome dos arquivos das imagens não deve conter espaços. Caso seja necessário, recomendamos que os caracteres sejam separados por hífen.
- Não utilize caracteres especiais nos nomes das imagens. Siga um  padrão alfanumérico.
- A extensão recomendada para os arquivos é .JPG (não .JPEG).
- Recomendamos que suas imagens tenham fundo branco.
- O formato das imagens deve ser preferencialmente quadrado, para evitar distorções.
- As imagens não podem ter menos de 600 pixels.
- As imagens podem ter até 3200 x 3200 pixels.
- Para que o zoom das imagens funcione corretamente, recomendamos que as imagens tenham a partir de 1000 pixels. Caso seja possível, você pode obter um resultado ainda melhor no zoom com imagens de 1500 a 2000 pixels.
- Recomendamos que as imagens não tenham mais do que 300 kb de tamanho.
- Não recomendamos esticar a imagem para não afetar sua qualidade.
