Na plataforma da VTEX, é possível exportar uma planilha com os [produtos cadastrados](https://help.vtex.com/pt/tutorial/cadastrando-produtos--tutorials_2567 "produtos cadastrados") em uma loja.  

A seguir, confira o passo a passo para realizar esta operação:

1. No Admin VTEX, acesse __Catálogo__, ou digite __Catálogo__ na barra de busca no topo da página.
2. Clique em __Importação e exportação__.
   Certifique-se que está na aba __Produtos e SKUs__.
3. Clique em __Exportação/Importação__.
4. Preencha o campo com o __email__ para o qual deseja enviar a planilha.
5. Clique em `Exportar para Excel`.

Em seguida, confira a caixa de entrada do email selecionado. Caso a operação tenha funcionado, o usuário terá recebido um email com um link para baixar a planilha completa.

>⚠️ O comando só funciona adequadamente em casos de até 10 mil produtos. Caso o valor requerido seja maior que esse limite, o sistema não envia a planilha para o email. Use os filtros para refinar sua busca.
