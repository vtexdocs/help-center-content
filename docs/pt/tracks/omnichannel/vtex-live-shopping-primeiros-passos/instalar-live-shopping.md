A solução **Live Shopping** é compatível com as seguintes tecnologias frontend:

* [Store Framework (VTEX IO)](/pt/tracks/trilha-da-loja-vtex--eSDNk26pdvemF3XKM0nK9/67SCtUreXxKYWhZh8n0zvZ#store-framework)
* [CMS - Portal Legado](/pt/tracks/cms--2YcpgIljVaLVQYMzxQbc3z/1oN446gRGcR2s70RvBCAmj)
* [Headless CMS](/pt/tutorial/headless-cms-overview--3U5gvhHdQL0jczYH8gjX09)
* [FastStore](https://developers.vtex.com/docs/guides/faststore/storefront-features-implementing-live-shopping-for-faststore)
* Lojas externas

## Pré-requisito

Exceto lojas externas, todas as soluções de frontend devem ter [o VTEX IO CLI](https://developers.vtex.com/docs/guides/vtex-io-documentation-vtex-io-cli-installation-and-command-reference) instalado.

## Obter o aplicativo VTEX Live Shopping

A menos que você seja uma loja externa, siga as etapas abaixo:

1. Acesse o [VTEX Live Shopping](https://apps.vtex.com/vtexventures-livestreaming/p) na VTEX App Store. Você também pode acessá-lo através do Admin VTEX em **Apps > App Store > VTEX Live Shopping**.
2. Clique em `Baixar App` para instalar o aplicativo. Faça login na sua conta VTEX, se necessário.
3. Clique em `Instalar App`.
4. Clique em `Instalar`.
5. Aceite os Termos de uso ao clicar em `Instalar`.

## Etapas de instalação para VTEX IO

Como o **VTEX Live Shopping** afeta a loja virtual exibindo um componente ao iniciar o evento, sua equipe técnica deve implementar a solução seguindo as instruções da [documentação para desenvolvedores](https://developers.vtex.com/docs/apps/vtexventures.livestreaming#for-vtex-io).

## Etapas de instalação para FastStore

Para integrar o **Live Shopping** a um projeto usando a [FastStore](https://developers.vtex.com/docs/guides/faststore), acesse o guia para desenvolvedores [Implementing Live Shopping for FastStore](https://developers.vtex.com/docs/guides/faststore/storefront-features-implementing-live-shopping-for-faststore).

> ℹ️ Essa funcionalidade está disponível como uma solução nativa para a última versão da FastStore. Se a sua loja usa [FastStore v1 ou v2](https://developers.vtex.com/docs/guides/faststore/getting-started-faststore-versions-and-support-levels), acesse o guia [Implementing Live Shopping for FastStore previous versions](https://developers.vtex.com/docs/guides/faststore/storefront-features-implementing-live-shopping-for-faststore-previous-versions).
