Uma solicitação de titular de dados é um pedido escrito que clientes da sua loja podem fazer para exercer seu direito de obter informações sobre os dados que estão sendo tratados pela loja e pela VTEX.

Qualquer pessoa tem o direito de confirmar se uma loja está tratando seus dados ou não. Se estiver, o titular também tem o direito de acessar os dados e saber a finalidade do tratamento, as categorias de dados pessoais e os destinatários ou categorias de destinatários com os quais seus dados pessoais foram compartilhados.

Para mais informações, confira o guia [Direitos dos titulares de dados pessoais](https://help.vtex.com/pt/tutorial/direitos-dos-titulares-de-dados-pessoais--6imchxTx09icupKMbzHVIM).

## Exclusão de dados

A qualquer momento, lojistas podem excluir dados pessoais a pedido dos clientes da sua loja. O cliente final deve fazer a solicitação diretamente ao lojista e essas solicitações devem ser encaminhadas pelo lojista à VTEX. O direito de exclusão se sobrepõe ao período de retenção previsto.

Se você gerencia uma loja VTEX e precisa de ajuda para excluir os dados dos usuários, leia [Exclusão de dados de clientes](https://help.vtex.com/pt/tutorial/exclusao-de-dados-de-clientes--1R9Fn7A06Ifj4R9YD4JTKU).

## Alteração de dados

Todos os dados pessoais de clientes da sua loja são editáveis na página [Minha Conta](https://help.vtex.com/pt/tutorial/como-funciona-a-minha-conta--2BQ3GiqhqGJTXsWVuio3Xh), exceto o email. Nesse caso, é possível excluir o perfil anterior e criar um novo perfil com outro email.
