Evite usar artigos em comandos, botões e menus, pois isso pode tornar o texto mais longo e menos direto.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| Adicionar produto | Adicionar um produto |

Use artigos para criar frases mais fluidas e claras quando essas frases forem mais longas, descritivas ou explicativas.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| Adicione um novo endereço de ponto de retirada | Adicione novo endereço de ponto de retirada |

O uso dos artigos definidos ou indefinidos deve ser consistente entre os elementos coordenados para manter o paralelismo sintático. Quando optar por usar o artigo em uma das partes, deve repeti-lo nas demais. Se decidir omitir, a omissão deve ser aplicada igualmente a todos os itens.

O gênero de siglas em português geralmente segue o substantivo principal que representam. No caso de SKU (Stock Keeping Unit), a tradução seria “unidade”, um substantivo feminino. No entanto, no contexto técnico da VTEX, adotamos a pronúncia /skjuː/ (“esquiu”) e tratamos SKU como um código técnico, utilizando a forma masculina: “um SKU”.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| Um SKU | Uma SKU |

Em português, é comum usar artigos definidos antes de nomes próprios para manter a fluidez, mas é necessário avaliar caso a caso.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>Acesse o Admin VTEX</li><li>A integração com o Intelligent Search</li></ul> | <ul><li>Acesse Admin VTEX</li><li>Acesse o Admin da VTEX</li><li>A integração com Intelligent Search</li></ul> |

Fique atento à estrutura da frase. O uso do artigo deve seguir a regência e estrutura natural da língua portuguesa.

| Correto ✅ | Incorreto ❌ |
| :--- | :--- |
| <ul><li>O pedido `{orderId}` será cancelado</li><li>Para mensagens de erro, que considera o espaço reduzido e a objetividade, isso poderia aparecer sem artigo, mas também sem verbo: Pedido `{orderId}` cancelado</li></ul> | <ul><li>Pedido `{orderId}` será cancelado</li></ul> |
