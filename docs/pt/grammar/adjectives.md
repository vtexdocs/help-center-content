Evite exageros e qualificativos vazios. Prefira adjetivos funcionais, mensuráveis ou verificáveis.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>API compatível com sistemas externos</li><li>Um fluxo mais rápido para aprovar alterações</li></ul> | <ul><li>API poderosa para todos os tipos de negócio</li><li>Um fluxo extremamente fácil e ágil</li></ul> |

Adjetivos devem ser evitados quando não agregam informação crucial e que contribua para o tom de voz da VTEX, de modo que não sejam usados adjetivos em excesso.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>Erro ao salvar as alterações</li><li>Configuração finalizada com sucesso</li><li>Pedido pendente de faturamento</li><li>Regras promocionais para sellers inativos</li></ul> | <ul><li>Ops, ocorreu um grave erro inesperado</li><li>Configuração finalizada com sucesso, agora sua loja está ainda mais atrativa para os clientes</li><li>Pedido parcial pendente de faturamento</li><li>Regras promocionais personalizadas válidas para sellers inativos</li></ul> |

Tenha atenção à mensagem que deseja passar ao posicionar adjetivos na frase.

| Exemplo |
| :--- |
| <ul><li>Grande projeto (importância) vs. projeto grande (tamanho)</li><li>Único cliente (exclusividade) vs. cliente único (com características únicas)</li></ul> |

Em geral, adjetivos devem vir após o substantivo, exceto quando se quer enfatizar uma qualidade subjetiva. Quando atuam como adjetivos, particípios também devem vir após o substantivo.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>Item removido</li><li>Erro inesperado</li><li>Recursos adicionais</li><li>Exemplo de exceções: Principais recursos, Melhores práticas</li></ul> | <ul><li>Removido item</li><li>Inesperado erro</li><li>Adicionais recursos</li><li>Exemplo de exceções: Recursos principais, Práticas melhores</li></ul> |

Há casos em que é possível usar nomes próprios como adjetivos para evitar usar preposição e tornar o texto mais direto. Os nomes devem vir direto após o substantivo e funcionar naturalmente na sua forma em português. A exceção é em nomes oficiais de produtos que permanecem sem tradução.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>Este é um produto VTEX desenvolvido para…</li><li>Blog da VTEX</li><li>Exemplo de exceção: VTEX Live Shopping</li></ul> | <ul><li>Este é um produto da VTEX desenvolvido para…</li><li>VTEX Blog</li><li>Exemplo de exceção: Live Shopping da VTEX</li></ul> |

Siga as regras de posição e pluralização do português quando o substantivo, o adjetivo ou ambos forem termos em inglês.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>Para incluir sellers white label na promoção…</li></ul> | <ul><li>Para incluir white label sellers na promoção…</li></ul> |

Embora o uso de hífen não seja característico do português para a forma substantiva, em contextos técnicos ou de marca, preservamos a forma original com hífen dos termos em inglês.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>Entrega last-mile</li><li>Funcionalidade all-in-one para lojas VTEX</li><li>Abordagem API-first</li></ul> | <ul><li>Entrega last mile</li><li>Funcionalidade all in one para lojas VTEX</li><li>Abordagem API first</li></ul> |

Nem todos os adjetivos variam em número seguindo o substantivo. Esteja atento às regras da língua portuguesa.

| ✅ Correto | ❌ Incorreto |
| :--- | :--- |
| <ul><li>Vales-presente</li><li>Caso você tenha várias contas franquia</li></ul> | <ul><li>Vales-presentes</li><li>Caso você tenha várias contas franquias</li></ul> |
