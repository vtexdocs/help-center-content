A última seção do módulo é onde o lojista pode configurar as formas de pagamento e os planos de recebimento de vendas do e-commerce.    

A primeira aba define as configurações das formas de pagamento. Já a segunda, a dos planos que o VTEX Payment oferece : Avulso, Antecipado D30 e Antecipado D40. 

Em ambos os casos, basta clicar no botão “Editar” que fica no canto superior direito de presente no box de cada configuração para definir o plano de recebimento.

Caso faça alguma alteração nas configurações mencionadas, nunca se esqueça de clicar no botão “Salvar’.
