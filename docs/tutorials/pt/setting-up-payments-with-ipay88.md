Você pode configurar o iPay88 para receber pagamentos na sua loja. Nesta modalidade, seu cliente escolhe o __meio de pagamento__ iPay88 para efetuar a compra.

Para configurar pagamentos com iPay88, siga os passos abaixo:

## Configurar afiliação de gateway iPay88
1. Acesse o módulo __Pagamentos__.
2. Clique em __Configurações__.
3. Na aba __Afiliações de Gateways__, clique no botão __+__.
4. Clique no conector __IPay88__.
5. No campo __MerchantCode__, preencha o código fornecido pelo iPay88.
6. Em __MerchantKey__, informe a chave fornecida pelo iPay88.
7. No campo __Currency code__, escolha o código de três letras que representa a moeda que será usada nas transações.
8. Clique em __Salvar__.

## Configurar condição de pagamento iPay88
1. Já dentro das __Configurações__ do módulo Pagamentos, clique na aba __Condições de pagamento__.
2. Clique no botão __+__.
3. Selecione o meio de pagamento __IPay88__.
4. Clique no botão __Status__ para ativar essa condição de pagamento.
5. O campo __Processar com a afiliação__ já vai estar preenchido pela opção _IPay88_.
6. Se houver um antifraude configurado na sua loja, você pode ativá-lo para essa condição de pagamento marcando a caixa __Usar Antifraude__.
7. Você pode configurar [condições especiais de pagamento](/pt/tutorial/condicoes-especiais).
8. Clique em __Salvar__.

Depois de seguir os passos indicados, o iPay88 vai aparecer no checkout da sua loja como uma opção de pagamento para seus clientes utilizarem.
