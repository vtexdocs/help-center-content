A maxiPago! é um [gateway](/pt/tutorial/o-que-e-um-gateway-de-pagamentos) que conecta sua loja aos principais adquirentes e meios de pagamento do Brasil. Para usar esse gateway na sua loja, recomendamos que você utilize o conector MaxiPagoV3, que é a integração mais atual.

Para configurar a maxiPago! com o conector MaxiPagoV3, siga os passos abaixo:

## Configurar afiliação de gateway MaxiPagoV3
1. Acesse o módulo __Pagamentos__.
2. Clique em __Configurações__.
3. Na aba __Afiliações de Gateways__, clique no botão __+__.
4. Clique no conector __MaxiPagoV3__.
5. No campo __Application Key__, informe a chave da aplicação, fornecida pela maxiPago!.
6. Em __Application Token__, preencha o campo com o token fornecido pela maxiPago!.
7. Clique em __Salvar__.

## Configurar condição de pagamento
Depois de seguir os passos indicados, o conector MaxiPagoV3 vai estar configurado na sua loja. Assim, quando você for cadastrar uma condição de pagamento, ele vai estar disponível no campo __Processar com a afiliação__ (desde que o meio de pagamento seja compatível com o MaxiPagoV3). Para saber como definir condições de pagamento, acesse [este artigo do Help](/pt/tutorial/condicoes-de-pagamento).
