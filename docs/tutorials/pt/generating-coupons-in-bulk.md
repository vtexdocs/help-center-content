A criação de cupons para promoções deve ser feita a partir do módulo **Promoções e Taxas** e está documentada em nosso artigo [Criar cupom](https://help.vtex.com/pt/tutorial/como-criar-cupom--tutorials_319). No entanto, podem ocorrer casos em que é necessário criar múltiplos cupons de uma vez, devido a alguma campanha ou ação de marketing.

A criação de cupons em massa não pode ser realizada pelo Admin VTEX, apenas pela Promotions & Taxes API. Veja como gerar cupons em grandes quantidades seguindo o guia [Criação e gerenciamento de cupons com a API de Promoções](https://developers.vtex.com/vtex-developer-docs/docs/creating-and-managing-coupons-with-promotions-api) (em inglês).
