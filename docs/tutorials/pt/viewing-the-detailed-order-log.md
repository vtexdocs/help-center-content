O fluxo de pedidos pode ser exibido através de diagramas ou interações. A primeira exibição é recomendada para que você tenha uma visão do processo como um todo, já a segunda para que você encontre algum erro ou detalhamento específico.

Para visualizar as interações de um pedido, siga os passos abaixo:

1. No Admin VTEX, acesse **Pedidos > Todos os pedidos**, ou digite **Todos os pedidos** na barra de busca no topo da página.
2. Clique no pedido desejado.
4. Clique no botão `Ver interações`.

Você verá as interações do sistma **Pedidos** com outros sistemas necessários para que um pedido seja concluído com sucesso, como **Logística**, **Promoção**, **Checkout**, **Pagamentos** e sistemas externos (ERPs, por meio de integração).

As interações mostram um log detalhado do pedido, com todas as interações feitas desde que o pedido é aceito até ser faturado. Elas são exibidas em ordem cronológica (de baixo para cima) e indicam, por exemplo, qual usuário aprovou, cancelou ou faturou um determinado pedido.

Desta forma, fica mais fácil visualizar a comunicação entre sistemas e entender os status e possíveis erros no fluxo de um pedido.
