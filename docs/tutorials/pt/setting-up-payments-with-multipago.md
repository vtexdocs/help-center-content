Na VTEX, é possível realizar a integração com o provedor de pagamentos Multipago. Com este conector, sua loja pode realizar vendas por meio de promissórias.

Para configurar a afiliação Multipago, acesse [Cadastrar afiliações de gateway](https://help.vtex.com/pt/tutorial/afiliacoes-de-gateway--tutorials_444#).

Para configurar o pagamento por meio de promissória, acesse [Configurar pagamentos com Promissória](https://help.vtex.com/pt/tutorial/configurar-pagamentos-com-promissoria#).

Depois de seguir os passos indicados, o conector Multipago pode demorar até 10 minutos para aparecer no checkout da sua loja como uma opção de pagamento.
