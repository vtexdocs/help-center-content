   </td>

   </td>
   <td class="t-body pa5" style="min-width: 15rem;">Permite que sellers visualizem a lista de SKUs enviados para os marketplaces em que vendem seus produtos. No Admin VTEX, a lista pode ser acessada em MARKETPLACE > Integrações > Produtos.
   </td>

   <td class="t-body pa5" style="min-width: 15rem;">Save Suggestion Rules
   </td>
   <td class="t-body pa5" style="min-width: 15rem;">UI resources
   </td>
   <td class="t-body pa5" style="min-width: 15rem;">Permite que marketplaces criem e alterem regras do VTEX Matcher para a aprovação de SKUs Recebidos. Essa configuração é feita por meio da API SKU Approval Settings. 
   </td>
  </tr>
  <tr class="bb b--muted-3">
   <td class="t-body pa5" style="min-width: 15rem;">Marketplace seller setup
   </td>
   <td class="t-body pa5" style="min-width: 15rem;">UI resources
   </td>
   <td class="t-body pa5" style="min-width: 15rem;">Permite associar o seller a um novo marketplace ou alterar uma configuração existente. Sellers incluídos no Seller Portal só poderão ter suas configurações alteradas, mas não associados a mais de um marketplace. Já sellers VTEX podem ser associados a outros marketplaces e terem suas configurações alteradas, ao habilitar esse recurso. Essa ação pode ser feita via Admin VTEX, ou via API Rest.
   </td>
  </tr>
