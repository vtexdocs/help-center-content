Authentication to VTEX APIs is provided through a pair of key (__appKey__) and token (__appToken__).

To learn how to create them, check the article [Creating appKeys and appTokens to authenticate integrations](/en/tutorial/creating-appkeys-and-apptokens-to-authenticate-integrations).
