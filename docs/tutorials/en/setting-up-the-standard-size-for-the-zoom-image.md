For the product zoom to work properly, you must set up the standard size for the zoom image. 

In the VTEX Admin, access **Store Settings > Storefront > Settings** and click *File Types*. By default, this type of zoom image is set up with 1000Ã—1000 pixels, which means that _the zoom will not work if you upload an image smaller than this._
