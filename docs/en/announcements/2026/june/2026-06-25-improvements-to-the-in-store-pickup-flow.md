VTEX is releasing improvements to the in-store pickup flow in the Last Mile module. Now, merchants can confirm order pickups more quickly using an alphanumeric token provided by the customer. After confirmation, the order invoice can be triggered automatically, depending on the configured integration.

The update reuses the VTEX email structure to send the confirmation token, making the _Buy Online Pickup In-Store_ (BOPIS) experience simpler and more seamless.

## What has changed?

Previously, merchants didn't have a simple way to confirm in-store pickups in Last Mile, which made it difficult to automatically generate the invoice and delayed the fulfillment flow.

Now, merchants can confirm order pickups directly in Last Mile using an alphanumeric token provided by the customer. After confirmation, the order invoice can be triggered automatically.

Key improvements include:

- Digital confirmation of in-store pickup directly in Last Mile.
- Using an alphanumeric token to validate order pickup.
- The order invoice can be triggered after pickup confirmation depending on the configured integration.
- Reusing the VTEX email structure to send the token to the customer.

## What needs to be done?

No action is required. The feature will be automatically provided to stores using the Last Mile module.
