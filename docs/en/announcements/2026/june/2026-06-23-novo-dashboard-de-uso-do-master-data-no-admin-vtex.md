As of this date, VTEX merchants now have access to the **Master Data usage** dashboard in the VTEX Admin. The new screen helps you track the volume of documents stored in [custom entities](https://help.vtex.com/en/docs/tutorials/master-data#custom-data-entities) of [Master Data](https://help.vtex.com/en/docs/tutorials/master-data) before making decisions about data storage or cleanup.

## What has changed?

The dashboard centralizes information about the usage of Master Data custom entities. The screen displays:

- **Total number of documents**: Total number of documents stored in the store's custom entities.
- **Last updated**: Date of the last update of the displayed data. This information is updated weekly.
- **Entities**: Table listing each custom entity and its number of documents in descending order of volume, with the highest amounts at the top.

The dashboard considers only custom entities and doesn't include the [Master Data native entities](https://help.vtex.com/en/docs/tutorials/master-data#master-data-native-entities), which are used by the platform's default features.

## What needs to be done?

No action is needed. The feature is already available in the VTEX Admin for you to check whenever you need to analyze the volume of documents in the store's custom entities.
