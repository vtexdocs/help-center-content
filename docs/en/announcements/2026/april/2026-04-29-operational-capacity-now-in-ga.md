The [Operational capacity](/en/docs/tutorials/operational-capacity-beta) module is now available for all VTEX stores. This feature is designed to help merchants manage sellers' capacity to receive and handle orders. This allows you to monitor order distribution among sellers and make adjustments to optimize the operational flow, enabling a more efficient logistics network.

![operational_capacity_image_1_PT](https://cdn.statically.io/gh/vtexdocs/help-center-content/refs/heads/main/docs/pt/tutorials/beta/envio-beta/capacidade-operacional-beta_1.png)

## What has changed?

Previously available only in Beta, the Operational Capacity feature now includes significant infrastructure improvements, such as the ability to segment capacity by [sales channel](/docs/tutorials/how-trade-policies-work), making management more flexible and efficient.

## What needs to be done?

No action is required, as all VTEX stores in all regions have already been migrated to the globally available version. To start using this feature, enable the module in the VTEX Admin under **Shipping > Operational capacity**.

> ⚠️ To use this feature, your [main account](/docs/tracks/accounts-and-architecture) must have linked [franchise accounts](/en/docs/tutorials/what-is-a-franchise-account).

For more details, see the [Operational capacity](/en/docs/tutorials/capacidade-operacional) article.
