We've updated the look and name of the Weni by VTEX platform, which is now called **VTEX CX Platform**. Effective March 26, this change provides a more integrated and consistent experience with the VTEX ecosystem.

## What has changed?

The main change is the new platform name, which is now **VTEX CX Platform**.

The menu structure has also been reorganized. Items are now grouped into two main sections: **Agent Builder** and **Operations**.

Other sections have also changed name and position in the menu. The table below shows the previous tool name, the new name, and where you can find it:

| **Old name**                         | **New name**          | **Location**          |
| ------------------------------------ | --------------------- | --------------------- |
| Insights                             | **Analytics**         | Homepage              |
| Agents                               | **My agents**         | Agent Builder section |
| Build — Content                      | **Knowledge base**    | Agent Builder section |
| Build — Manager and profile settings | **Edit manager**      | **My agents** page    |
| Build — Instructions settings        | **Edit instructions** | **My agents** page    |
| Flows                                | **Automation flow**   | Agent Builder section |
| Human support                        | **Live Desk**         | Operations section    |
| Conversations                        | **Audit**             | Operations section    |
| Apps                                 | **Channels**          | **Settings** page     |
| Studio                               | **Contacts**          | Operations section    |
| Send templates                       | **Campaigns**         | **Contacts** page     |

## Why did we make this change?

We developed this update to create visual consistency with the VTEX identity, unifying the user experience across different VTEX products. The main advantages are:

- **Visual consistency:** Interface aligned with the VTEX pattern and identity, providing a unified experience.
- **More intuitive navigation:** The new section grouping makes it easier to find the right tools quickly.
- **More accessible names:** The new section names better reflect the function of each element, reducing the learning curve.

## What needs to be done?

No action is needed. The update is automatic and already available to all users.

To learn more about VTEX CX Platform and its features, check our [documentation](https://help.vtex.com/en/docs/tutorials/vtex-cx-platform-category).
