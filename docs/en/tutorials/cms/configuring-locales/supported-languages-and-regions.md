When [creating a locale](/en/docs/tutorials/configuring-locales) for your store, you must select a language and a region. The platform supports a predefined set of languages and regions that can be combined to form locale codes, such as `en-US` or `pt-BR`.

The following technical specifications apply to all supported locales:

* Encoding: UTF-8  
* Text direction: Left-to-right (LTR)  
* Supported scripts: Latin, Cyrillic

Below is the complete list of supported languages and regions.
