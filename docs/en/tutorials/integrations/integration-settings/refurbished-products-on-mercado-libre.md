The **Refurbished products** feature in the VTEX and Mercado Libre integration allows sellers to publish and sell products that were inspected, cleaned, or repaired by specialists to ensure they work properly.

## How to activate the feature

To activate the feature, the seller must **create a customizable field** in VTEX Admin and **enable the setting** in the Mercado Libre integration card.

### 1. Create the customizable field in VTEX Admin

Follow the instructions in [Adding SKU specifications or fields](/en/tutorials/catalog/products-and-skus/adding-sku-specifications-or-fields) to create a SKU field.

When creating the field, make sure it is available for the SKUs that will be sent to Mercado Libre.

### 2. Enable it in the integration card

After creating the field, enable the feature in the Mercado Libre integration card:

1. Go to **VTEX Admin > Marketplaces and integrations > Mercado Libre**.
2. In the integration card, find the **Refurbished products** section.
3. Select the checkbox.

With the feature enabled, the seller must fill in the SKU field with the product condition so the information can be sent correctly to Mercado Libre.
With the feature enabled, the seller must fill in the SKU field created with the product condition so the information can be sent correctly to Mercado Libre, which accepts the **Refurbished status** attribute with only the following values:

- `Excellent`
- `Good`
- `Fair`

After enabling the feature, you must complete the mapping in Mercado Libre so the product is published as refurbished.
