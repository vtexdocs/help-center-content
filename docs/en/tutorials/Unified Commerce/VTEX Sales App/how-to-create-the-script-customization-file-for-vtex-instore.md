To create the customization file, access the Portal through the topbar and create a new file named `checkout-instore-custom.js` This file will be loaded by inStore automatically. 

![topbar-portal](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/en/tutorials/Unified%20Commerce/VTEX%20Sales%20App/how-to-create-the-script-customization-file-for-vtex-instore_1.png)

Open the Portal module and click on the button to edit the default site.  In the example below, the storeâ€™s accountname is  __Omniera__.

![portal-default-website](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/en/tutorials/Unified%20Commerce/VTEX%20Sales%20App/how-to-create-the-script-customization-file-for-vtex-instore_2.png)

Then, click on the `Code` tag, and on the `New` button to select the option for a new `File`.

![portal-add-file](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/en/tutorials/Unified%20Commerce/VTEX%20Sales%20App/how-to-create-the-script-customization-file-for-vtex-instore_3.png)

Enter the `checkout-instore-custom.js` file name, and click on `Create`.

![portal-add-file-filename](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/en/tutorials/Unified%20Commerce/VTEX%20Sales%20App/how-to-create-the-script-customization-file-for-vtex-instore_4.png)

Now you can add the file content and save it.  inStore will load the customization file automatically. 

![portal-edit-file](https://raw.githubusercontent.com/vtexdocs/help-center-content/refs/heads/main/docs/en/tutorials/Unified%20Commerce/VTEX%20Sales%20App/how-to-create-the-script-customization-file-for-vtex-instore_5.png)
