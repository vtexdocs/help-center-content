When you start using VTEX, your company receives an **account** that gives access to the Admin, where you can manage all aspects of your ecommerce operation. This account is the central environment to configure and operate your online store.

## Access the VTEX Admin

You can access the Admin via browser at the following URL, where `{accountName}` should be replaced by [your account name](/docs/tutorials/what-is-account-name):

```
{storename}.myvtex.com/admin
```

After logging in, you'll have access to your store's exclusive admin dashboard, where you can manage products, orders, customers, settings, and more.

## Account settings

To access **Account settings**, click your profile avatar — indicated by your email initial letter — in the upper right corner of the Admin, and then click **Account settings**.

![Admin v4 account settings menu PT](https://cdn.statically.io/gh/vtexdocs/help-center-content/refs/heads/main/docs/pt/tutorials/gerenciamento-da-conta/visao-geral-configuracoes-da-conta_1.jpg)

In this section, you'll find the main pages to manage your account:

### Account

Page where you can view and manage your VTEX account details, including:

- Company information (name, company registration number, logo)
- Account and platform status
- Store and multistore management
- Domain configuration

> ℹ️ See [Account management](/docs/tutorials/account-management) for more details.

### Users

Page to create and manage administrative users who have access to your store's Admin. On this page, you can:

- Create new users
- Edit permissions of existing users
- Delete users
- Export user list

> ℹ️ Learn more in [Managing administrative users](/docs/tutorials/managing-admin-users).

### User roles

Page to create and manage user roles, which determine which features each user group can view and edit on the platform.

> ℹ️ Learn more in [Roles](https://help.vtex.com/docs/tutorials/roles).

### Authentication

Page to configure the available login methods for store customers and administrative users, including:

- Access code
- Password (with expiration option)
- Login with Google, Facebook
- Integration with external identity providers (OAuth, SAML)

> ℹ️ Learn more in [Authentication](/en/docs/tutorials/authentication).

### API keys

Page to manage API keys (app keys and app tokens) used to authenticate integrations with external systems via API. These keys ensure secure access to data without exposing your account to unauthorized applications.

> ℹ️ Learn more in [API keys](/en/docs/tutorials/api-keys).
