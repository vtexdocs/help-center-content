The page template is a model designed to define an HTML structure for displaying page content.

You can use this template to reference the JS and CSS files that will be used on a specific page, as well as other templates, like shelf templates or subtemplates.

In addition, page templates are also responsible for including components such as banners, suggestion shelves and other features available on the platform.

At VTEX, the user has the option of either creating new templates or editing existing templates.

To create a new template, we recommend that you use a default template as a base.

Check out the walk-through:

1. Access the CMS module.
2. Click on __Layout__.
3. Click on the CMS folder.
4. Click on __HTML Templates __.
5. Just below, click on __New template__.
6. Fill in the __Template Name__ field.
>❗ When filling in the **Template Name**, avoid using the following reserved words, as this can hinder your store's performance and interfere with the functioning of VTEX Admin pages. See the list of reserved words below.
>
> *`a`
>
> *`meta`
>
> *`api`
>
> *`admin`
> 
7. Then fill in the __Template XHTML__ box with your template code. 
![4 1](https://images.ctfassets.net/alneenqid6w5/1BJRfbclbjHPUSztB6y0kP/bd95af131b622a00d9b41dee47789987/4_1.png)
9. Click on the __Save Template__ button.
![4 2](//images.ctfassets.net/alneenqid6w5/58ZckRTKZ2e6BbXPtx54xK/a638e74bf68dc437257a6c220d5a2c1c/4_2.png)

To edit an existing template, the process is almost identical.

But instead of clicking on the New Template option, you must click on the template you want to edit.

Check out the complete walk-through below:

 1. Access the CMS module.
 2. Click on __Layout__.
 3. Click on the __CMS__ folder.
 4. Click on the desired template option.
 5. Make the necessary changes.
 6. Click on the __Save Template__ button.
