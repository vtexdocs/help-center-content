A instalação pelo VTEX IO é bem simples - basta instalar o aplicativo pela VTEX Store e realizar as configurações necessárias seguindo esta documentação. Para iniciar a indexação é necessário acessar a funcionalidade [Configurações de Integração](https://help.vtex.com/tracks/vtex-intelligent-search--19wrbB7nEQcmwzDPl1l4Cb/6wKQgKmu2FT6084BJT7z5V). Após esse passo, o sistema se iniciará e todo o layout implementado dentro do IO será reaproveitado.

Para começar a usar o VTEX Intelligent Search, é preciso instalar `vtex.admin-search` e `vtex.search-resolver@1.x` no [workspace](https://vtex.io/docs/concepts/workspace/) desejado como mostra a linha abaixo:


`vtex install vtex.admin-search vtex.search-resolver@1.x`


O VTEX Intelligent Search apresenta também componentes adicionais para o Admin. Caso você também queira instalar esses componentes, acesse a seção UI Components da [nossa documentação do VTEX IO](https://vtex.io/docs/app/vtex.search@1.0.3/).
