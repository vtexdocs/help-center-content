Order integration errors may occur in marketplaces for which VTEX has developed a [native connector](https://help.vtex.com/en/tutorial/estrategias-de-marketplace-na-vtex--tutorials_402#integrating-with-a-native-connector-vtex). If that happens, you will be able to see a message that identifies the nature of the error. You can find error messages in the VTEX Admin by accessing **Marketplace > Connections > Orders**, and clicking on the order.

These errors are commonly related to one of the following situations:

- [SLA errors](https://help.vtex.com/en/tutorial/erros-de-sla-na-integracao-de-pedidos-de-marketplace--X8lSfxT44OyxkxwvnRk1X)
- [Out-of-stock errors](https://help.vtex.com/en/tutorial/erros-de-falta-de-estoque-na-integracao-de-pedidos-de-marketplace--s1i5OCcPFslrMkZJLDnfP)
- [Price divergence errors](https://help.vtex.com/en/tutorial/resolucao-de-erros-de-divergencia-de-preco-em-pedidos-de-marketplace--6MbmPX4SKyRkcTJxVhRna8)

To see the most common order integration error messages in each marketplace and how to solve them, click on the name of the marketplace:

- [Amazon](https://help.vtex.com/en/tutorial/erros-de-integracao-de-pedidos-da-amazon--QCOquR8cai882HhDOqNm7)
- Mercado Libre (LATAM only)
- Dafiti (LATAM only)
- Magazine Luiza (Brazil only)
- B2W (Brazil only)
- Via (Brazil only)
- Netshoes (Brazil only)
- Carrefour (Brazil only)
